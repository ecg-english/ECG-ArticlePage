# ECG Community Hub

ECG留学空間のコミュニティ記事ページです。

## セットアップ

1. 依存関係のインストール:
```bash
npm install
```

2. 開発サーバーの起動:
```bash
npm run dev
```

3. ブラウザで `http://localhost:5173` を開いてください。

## ビルド

本番環境用のビルド:
```bash
npm run build
```

ビルドされたファイルは `dist` フォルダに生成されます。

## 技術スタック

- React 18
- Vite
- Tailwind CSS
- Lucide React (アイコン)
